# Rider XAML Style Dependency Visualizer {kDg}

This repository contains a JetBrains Rider plugin that analyzes dependencies between styles in XAML files. The plugin visualizes these relationships using diagrams generated by a PlantUML server.

## Features

- Scans XAML files for style definitions and their dependencies
- Generates dependency diagrams using PlantUML
- Integrates directly into Rider for seamless workflow
- Possibility to zoom and pan the generated diagram for better navigation

## Requirements

- JetBrains Rider
- PlantUML server (local or remote)

## Usage

1. Install the plugin in Rider.
2. Right click on XAML file.
3. Tools -> kDg Xaml Style Diagram.
4. The diagram opens in the Xaml Style Diagram tool window (main menu → View → Tool Windows).

## Configuration

Configure the plugin in settings by specifying:
- The PlantUML server URL to generate diagrams.
- A custom PlantUML diagram header for styling.
- The diagram type to choose between class or object dependency visualization.
